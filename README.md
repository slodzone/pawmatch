# PawMatch

A Tinder-style pet-sitter discovery prototype: owners register, then swipe
through sitter profiles to match.

This is a frontend-only prototype (no backend yet) — registration and matches
exist only in browser memory and reset on refresh.

## Local development
```
npm install
npm run dev
```

## Deployment
Deployed via Vercel, auto-building from this repo's `main` branch.
